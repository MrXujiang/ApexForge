- 组件统一使用 `React.forwardRef` + `displayName` 暴露 ref，并通过 `asChild` + `Slot` 支持根元素透传以适配 Radix 组合场景。
- 组件样式变体通过 `cva(...)` 声明 `variants` 与 `defaultVariants`，在组件内部调用 `buttonVariants({ variant, size, className })` 再经 `cn()` 合并输出。
- 所有跨层数据结构集中在 `types/` 下按业务域拆分文件，并由 `api/client.ts` 等基础设施通过泛型 `ApiResponse<T>` 复用，避免重复定义。
- 工具函数保持纯函数形态且无外部副作用，如 `validatePrompt` 返回 `null | string` 表示校验结果，便于与表单库集成。
- 主题与配色常量以对象字面量导出（`theme.palette`、`modelThemePresets`），而非 CSS 变量或运行时计算，保证构建期可分析。
