- 对外统一返回 `{ traceId, data }` 包裹结构，traceId 通过 `createTraceId()` 生成，便于全链路追踪。
- JSON 字段在入库前用 `stringifyJson` 序列化、读取后用 `parseJson` 反序列化并提供默认值，避免空 JSON 字符串污染查询。
- 分类推断采用“关键词匹配优先 → 正则抽取短语 → 回退 product”的分层策略，并在多处复用同一组中英文关键词数组。
- 可变参数通过种子 `variantSeed = Date.now() % 100000 + random` 派生，再用取模映射到有限枚举集合（如 mirrorVariant/frameStyle），保证可复现的变体空间。
- 数值型意图字段在合并后一律用 `clamp(value, min, max)` 限幅，防止 LLM 输出溢出业务阈值。
