本模块是 ApexForge API 的启动与基础设施边界，由四个部分组成：
- `main.ts`：NestJS 应用入口，通过 `NestFactory.create` 创建实例，启用 CORS、全局前缀 `api/v1` 和带 `whitelist`/`transform`/`forbidNonWhitelisted` 的 `ValidationPipe`，从环境变量 `API_PORT` 读取监听端口。
- `app.module.ts`：根模块，使用 `ConfigModule.forRoot({ isGlobal: true })` 注入配置，并聚合业务子模块（Health、Templates、Validation、Generation、Assets、Feedback）与 `PrismaModule`。
- `prisma/prisma.module.ts` + `prisma.service.ts`：将 PrismaClient 扩展为 `PrismaService`，实现 `OnModuleInit`/`OnModuleDestroy` 钩子管理 `$connect`/`$disconnect`；模块标注 `@Global()` 后导出，使所有子模块无需重复 import 即可注入 `PrismaService`。
- `common/json.ts`：纯函数库，提供基于时间戳+随机数的 ID 生成器（`createId`/`createTraceId`）以及安全的 JSON 序列化/反序列化包装（空值回退）。
