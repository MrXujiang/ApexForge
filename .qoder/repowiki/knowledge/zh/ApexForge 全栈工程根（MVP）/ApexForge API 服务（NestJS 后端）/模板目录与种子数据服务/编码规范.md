- 控制器方法统一返回 `{ traceId: createTraceId(), data: ... }` 包装结构，便于链路追踪。
- JSON 字段（如 tags）通过 `stringifyJson` / `parseJson` 工具函数在持久化层序列化与反序列化，避免手写 JSON 操作。
- 分类归一化采用关键词包含匹配（中英双语词表），由独立 `normalizeTemplateCategory` 函数集中处理，查询前统一转换后再传给 Prisma where。
- 种子数据以纯常量数组 + `onModuleInit` + `upsert` 的模式保证幂等初始化，不在迁移脚本中硬编码。
